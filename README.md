Sistem Klasifikasi Hoax berbasis Web Single Page Application menggunakan machine learning classification untuk analsisi struktur bahasa dan membandingkan dengan sumber terpercaya dengan cosine similarity

Tech Stack yang digunakan
1. Frontend
  a) Bun
  b) React
  c) Lucide-React
  d) Tailwindcss
2. Backend
  a) Regex
  b) String
  c) Torch
  d) HuggingFace Transformers
  e) Requests
  f) Tesseract
  g) Pytesseract
  h) Fastapi
  i) Pillow
  j) Yake
  k) io
3. Model Training
  a) Kagglehub
  b) Torch
  c) HuggingFace Transformers
  d) HuggingFace Datasets
  e) HuggingFace Evaluate
  f) Pandas
  g) Numpy
  h) OS
  i) String
  j) Regex
  k) Scikit-learn
  l) Indobert-base-p2 by indobenchmark
