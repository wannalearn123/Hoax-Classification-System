Sistem Klasifikasi Hoax berbasis Web Single Page Application menggunakan machine learning classification untuk analsisi struktur bahasa dan membandingkan dengan sumber terpercaya dengan cosine similarity

Tech Stack yang digunakan
1. Frontend
  - Bun
  - React
  - Lucide-React
  - Tailwindcss
2. Backend
  - Regex
  - String
  - Torch
  - HuggingFace Transformers
  - Requests
  - Tesseract
  - Pytesseract
  - Fastapi
  - Pillow
  - Yake
  - io
3. Model Training
  - Kagglehub
  - Torch
  - HuggingFace Transformers
  - HuggingFace Datasets
  - HuggingFace Evaluate
  - Pandas
  - Numpy
  - OS
  - String
  - Regex
  - Scikit-learn
  - Indobert-base-p2 by indobenchmark
